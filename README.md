# Value Watts Dual-Channel Power Meter Comparison Tool
I built this tool because I believe cycling performance and tech insights should be accessible to everyone—without the expensive price tag. This is a lightweight desktop dashboard designed to run real-time, side-by-side telemetry comparisons for two cycling power sources. It’s simple, fast, and gets straight to the point.

# What It Does
1. It pulls in your power source data  via Bluetooth and compares it to another power source, wether it's a smart trainer, power meter spider, pedals...
2. The dashboard gives you an instant, color-coded visual check:
🟢 Green (< 5.0% discrepancy): You're golden. Typical drivetrain efficiency loss or solid calibration.
🟡 Yellow (5.0% - 9.9% discrepancy): Something is drifting. Time to run a spin-down calibration, adjust roller tension, or check for tire micro-slippage.
🔴 Red (>= 10.0% discrepancy): Total data desync or a massive calibration error. Time to pull over and troubleshoot.
